# Day 1 — Getting Started with Claude and Cowork

**Dates:** May 18, June 1, and June 8, 2026
**Time:** 1:00–2:30pm (optional work time 2:30–4:00pm)
**Location:** Learning Lab Studio, 50 Church St., Suite 374 / 3rd Floor

## Intended Audience

Faculty with no prior experience or limited experience with Claude, the Claude desktop app, or Claude Cowork. Faculty who already use Claude Cowork or Claude Code regularly may wish to skip this session.

## Description

Day 1 is an orientation to Claude's three interfaces:

1. The **chat window** most faculty already know.
2. **Cowork** — an interface that allows you to give Claude read and write access to files on your computer.
3. **Claude Code** — a terminal-based tool that takes the abilities of Cowork even further.

The majority of the session is devoted to Cowork. We'll install the desktop app, tour the interface, work through the basics of prompting, and connect Cowork to a folder of course materials or research data so you leave with a working setup on your own machine. We'll close with a brief explanation of how Cowork and Claude Code differ and a demonstration of what Claude Code and more advanced techniques can enable.
