# Day 3 — Advanced Claude Code: Skills, MCPs, Hooks, and Multi-Agent Workflows

**Dates:** May 20, June 3, and June 10, 2026
**Time:** 1:00–2:30pm (optional work time 2:30–4:00pm)
**Location:** Learning Lab Studio, 50 Church St., 3rd Floor

## Intended Audience

Faculty who have their Claude Code development environment set up and working, and are ready for more advanced techniques.

## Description

This session is for faculty who have already installed Claude Code and are ready to go beyond the basics. We'll work through:

- **Skills** — folders Claude Code discovers automatically when a task pattern matches.
- **MCP integrations** — connections to Google Drive, GitHub, databases, and other services.
- **Custom slash commands and hooks.**
- **Design patterns** for structuring a context folder so all of these pieces work together.
- **Multi-agent workflows** — when complex agent orchestration is worth the setup, and what kinds of problems it actually solves.

We will also discuss how to incorporate third-party AI tools using APIs.
