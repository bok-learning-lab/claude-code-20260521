Venn Diagram Explorer — multi-instance test

Three independent widgets on one page. Each must keep its own state: selecting an expression in widget 1 must not change the highlight in widgets 2 or 3, the mode toggle must scope to its own diagram, the "Clear highlight" button only affects the widget it belongs to, and the SVG region paths must render correctly on every diagram (verifying that the geometry computations don't depend on shared state). Widget 2 starts in three-set mode and only allows three-set mode (no toggle). Widget 3 omits the title and uses a custom expression list.
